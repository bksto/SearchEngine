Browsers: IE Firefox     You are seeing this page because you have disabled scripts in your browser or have a browser that doesn’t support client-side scripting. IE (Internet Explorer) To make sure your javascript is enabled on IE, go to the Tools menu -> Internet Options -> Security tab -> Custome Level... -> near the bottom there are radio buttons for "Active Scripting", make sure the radio button for Enable is selected. Firefox Make sure your javascript is enabled on Firefox. Go to Tools menu item -> Content tab -> check the box that says "Enable Javascript".
