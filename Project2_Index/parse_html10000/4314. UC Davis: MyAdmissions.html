MyAdmissions allows you to check the status of your UC Davis application and, if you are admitted, complete your required pre-enrollment tasks. Before entering the site, you must set up a UC Davis Login ID and password using the "Set Up Account" button at right. If you have lost your ID or passphrase, please visit the forgot passphrase page to use the online reset system. If you are still unable to login, please review our Frequently Asked Questions.
