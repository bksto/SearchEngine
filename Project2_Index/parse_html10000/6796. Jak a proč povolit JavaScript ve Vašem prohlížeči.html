Zde můžete najít instrukce jak povolit (aktivovat) JavaScript v pěti nejpoužívanějších prohlížečích. Internet Explorer Mozilla Firefox Google Chrome Opera Apple Safari Ve Vašem webovém prohlížeči je Javascript zakázán. Pokud Javascript povolíte, tento text se změní.
